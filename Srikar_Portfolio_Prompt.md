### Spacing System
Use Tailwind's default spacing scale. Section vertical padding: `py-24 lg:py-32`. Card padding: `p-6 lg:p-8`.

### Glassmorphism Card Style
```css
.glass-card {
  background: rgba(255, 255, 255, 0.7);  /* dark: rgba(255,255,255,0.04) */
  backdrop-filter: blur(12px) saturate(180%);
  -webkit-backdrop-filter: blur(12px) saturate(180%);
  border: 1px solid rgba(0, 0, 0, 0.08); /* dark: rgba(255,255,255,0.08) */
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.06); /* dark: 0 4px 24px rgba(0,0,0,0.4) */
}
```

### Border Radius
- Cards: `rounded-2xl` (16px)
- Buttons: `rounded-xl` (12px)
- Tags/Badges: `rounded-full`
- Images: `rounded-2xl`

### Shadows
```
sm:  0 1px 3px rgba(0,0,0,0.06), 0 1px 2px rgba(0,0,0,0.04)
md:  0 4px 16px rgba(0,0,0,0.06), 0 2px 8px rgba(0,0,0,0.04)
lg:  0 8px 32px rgba(0,0,0,0.08), 0 4px 16px rgba(0,0,0,0.04)
xl:  0 20px 60px rgba(0,0,0,0.10), 0 8px 24px rgba(0,0,0,0.06)
```

### Animation Principles
- Duration: 200–400ms for micro-interactions, 500–700ms for section reveals
- Easing: `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out-expo)
- Entrance: fade-up (translateY: 20px → 0, opacity: 0 → 1), triggered by Intersection Observer
- Hover: scale(1.02) on cards, background tint on buttons
- Stagger: 80ms between list items
- NO bounce, spring physics only for scroll-triggered counters
- All animations wrapped in `useReducedMotion()` guard
